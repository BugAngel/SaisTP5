  关于我们

这是一款基于Web的留学信息管理和分析平台，提供大量留学相关信息、利用智能算法的使用加强留学院校推荐的分析与匹配的专业性，建立良好的用户间互动关系。 此平台具有以下功能：拥有大量不同国家的不同院校供用户参考，用户可以登录自己的账号或创建账号，能够创建并且修改自己的相关资料，后台根据算法进行计算，进行给用户智能推荐或匹配并展示给用户，用户也可以在论坛与他人进行留学相关的交流；管理员用户可以实时更新院校信息，如修改院校信息、添加新的院校等，也可以对论坛进行管理，对于恶意、不相关的帖子进行删除，对恶意的用户进行论坛的禁言、封禁等； 此平台致力于帮助想要出国的学生，提供了大量信息、智能推荐和分析、论坛交流等免费服务，相比与如今各种中介网站，提供了更多有效的信息，网页简洁，一目了然。 本项目的主要特色是提供大量信息、智能推荐、社区交流的开放平台，使得在花费最少资源（时间、空间）的情况下使得用户获取想要的留学信息。
