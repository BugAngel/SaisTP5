  使用方法

在登录前，用户可以使用以下功能：

查看不同国家不同学校的简略信息；

根据关键词搜索院校（按国家、排名等）；

点击查看相应学校的详细信息；

查看论坛；

获得学校的入学条件；

获取学校的入学材料和申请流程

在登录后，用户可以额外使用以下功能：

更新自己的资料；

搜索院校时自动更新用户画像；

可以在论坛发帖与留言；

系统智能推荐院校；

进行智能评估；

管理员用户可以对论坛进行管理
